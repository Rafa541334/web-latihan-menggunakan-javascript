🥗 Appetizer: HTML (Struktur Restoran)

HTML adalah pondasi website. Bayangkan sebagai bangunan, lantai, meja, kursi.

  • Membuat struktur halaman
  • Menampilkan teks, gambar, video
  • Menyediakan form interaktif
  • Menyisipkan media
<h1>Selamat Datang</h1>
<p>Belajar HTML</p>
<button>Klik</button>

🍝 Main Course: CSS (Dekorasi Restoran)

CSS mengatur tampilan. Bayangkan sebagai cat dinding, dekorasi, pencahayaan.

  • Mengatur warna, font, shadow
  • Layout responsif: flex, grid
  • Membuat animasi: hover, fade
h1 {
color: gold;
text-align: center;
}
button {
background: black;
color: white;
padding: 10px 20px;
border-radius: 5px;
}

🍰 Dessert: JavaScript (Pelayanan Restoran)

JavaScript membuat website interaktif, seperti waiter melayani pelanggan.

  • Menangani klik, hover, submit, input
  • Manipulasi DOM
  • Membuat animasi & validasi form
// Klik tombol
const button = document.querySelector('button');
button.addEventListener('click', function(){
alert("Pesanan diterima!");
});
